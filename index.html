ArcGIS Online • R • Environmental GIS

California Wildfire Spatial Analysis — 2013–2019

This interactive GIS project analyzes California wildfire patterns from 2013–2019. Wildfire incident data was processed in R, summarized by county, and joined to California county boundaries in ArcGIS Online to visualize wildfire frequency, total acres burned, and county-level wildfire severity.

Workflow

  • Cleaned and summarized California wildfire incident data using R.
  • Created county-level statistics for fire frequency, acres burned, average burn size, and fatalities.
  • Joined wildfire summary data to California county polygons in ArcGIS Online.
  • Built interactive choropleth maps with county-level pop-ups and wildfire severity styling.

What this shows

The project highlights spatial patterns in wildfire frequency and burn severity across California, showing how GIS and statistical analysis can support environmental hazard interpretation.

ArcGIS Pro • Environmental GIS

Wildfire Risk and Land Cover Analysis — Lake Tahoe Region

This project combines wildfire perimeter data, NLCD land cover, elevation, OpenStreetMap roads, and water features to visualize how wildfire activity relates to vegetation, terrain, and human infrastructure around Lake Tahoe.

Workflow

  • Acquired raster and vector datasets from public GIS sources.
  • Prepared layers in ArcGIS Pro and organized them by map purpose.
  • Symbolized wildfire, water, roads, land cover, and elevation for readability.
  • Built a final cartographic layout with scale bar and north arrow.

ArcGIS Online • Climate & Habitat Mapping

Clarkia Population Climate Suitability Mapping

This ArcGIS Online project maps Clarkia population sites across California and compares annual precipitation, elevation, and temperature variables to evaluate ecological patterns tied to plant distribution.

Workflow

  • Mapped Clarkia population locations in ArcGIS Online.
  • Displayed climate and elevation values using labeled point data.
  • Used symbology to visualize precipitation differences across sites.
  • Created a web GIS visualization focused on plant-environment relationships.
