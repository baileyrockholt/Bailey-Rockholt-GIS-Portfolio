# Bailey Rockholt GIS Portfolio

Live portfolio site for environmental GIS work, including ArcGIS Pro projects, wildfire mapping, land cover analysis, and spatial data preparation.

## How to update
Replace the images in the `images` folder with exported GIS project screenshots. Keep the same filenames or update the paths in `index.html`.

- `tahoe-final-map.jpg`
